# Portfolio – Mario Mocholí Rubio

Portfolio personal desarrollado para presentar mi perfil como desarrollador web en formación. Este proyecto recoge mi formación académica, conocimientos técnicos y algunos trabajos representativos.

---

## Sobre mí

Soy estudiante de Desarrollo de Aplicaciones Web con formación previa en Microinformática y Redes. Me caracterizo por ser una persona organizada, constante y con una fuerte orientación al aprendizaje continuo.

Tengo especial interés en el desarrollo backend y en la estructuración correcta de aplicaciones, aunque también valoro la importancia de un frontend claro, funcional y bien diseñado. Mi objetivo es desarrollar soluciones mantenibles, escalables y técnicamente sólidas.

Además de la programación, soy un apasionado de la música y compongo en mi tiempo libre. Esta faceta creativa complementa mi perfil técnico y refuerza mi disciplina y capacidad de concentración.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- Diseño responsive con Flexbox y Grid
- Variables CSS
- Estructura semántica y buenas prácticas de maquetación
- Control de versiones con Git, GitFlow y uso de Forks para colaboración entre dispositivos y equipos
---

## Estructura del proyecto

El portfolio incluye las siguientes secciones:

- Carta de presentación
- Conocimientos técnicos (Frontend, Backend, Sistemas y Redes)
- Proyectos destacados
- Formación académica
- Información de contacto
